My fourth project with React.js

It's a simple recipe search engine. 

GitHub Pages:
https://adamkulig.github.io/recipes-search-engine-react-redux/

I used:
- Create-react-app 
- React.js 
- Redux
- Redux-thunk
- React-router
- SCSS (RWD)
- Javascript (ES6) 
- CSS Grid
- Api from edamam.com

Capabilities:
- fetch recipes
- adding to favorites
- deleting from favorites
- showing details
- auto-loading recipes after scroll down
